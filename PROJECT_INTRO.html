Tiga-Todo 项目介绍

一个现代化的待办事项应用，支持 Markdown 文档、实时同步、多端适配

项目概述

Tiga-Todo 是一款现代化的跨平台待办事项应用，采用"任务优先笔记"的设计理念。

技术栈

前端

  • React 18 + Vite + TypeScript
  • Framer Motion 动画
  • Tiptap 富文本编辑器
  • Tailwind CSS + Capacitor

后端与服务

  • Supabase（数据库、认证、实时同步）
  • Vercel 部署

功能特性

已完成

  • 用户注册/登录/登出
  • 任务 CRUD
  • 本地通知提醒
  • Glassmorphism UI + Spring 动画
  • Markdown + 代码高亮 + Mermaid

待开发

  • 任务拖拽排序、搜索、标签
  • 子任务、重复任务
  • Dark Mode
  • 数据导出、日历视图

项目结构
